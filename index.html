¿Qué es Flutter?

Flutter es un framework desarrollado por Google, que a finales de 2018, fue publicado como un proyecto de código abierto. El objetivo de este framework es crear aplicaciones nativas de forma fácil, rápida y sencilla. Su principal ventaja radica en que genera código 100% nativo para cada plataforma, con lo que el rendimiento y la UX es totalmente idéntico a las aplicaciones nativas tradicionales.

Caracteristicas

Hecho por Google

Flutter es el kit de herramientas de UI de Google para realizar hermosas aplicaciones, compiladas nativamente, para móvil, web y escritorio desde una única base de código.

Desarrollo Rápido

Trae tu aplicación a la vida en cuestión de milisegundos con Hot Reload. Utilice un completo conjunto de widgets totalmente personalizables para crear interfaces nativas en cuestión de minutos.

Rendimiento Nativo

Los widgets de Flutter incorporan todas las diferencias críticas entre plaformas, como el scrolling, navegación, iconos y fuentes para proporcionar un rendimiento totalmente nativo tanto en iOS como en Android.

Compatibilidad
con sistemas

Android

En Android, el View es la base de todo lo que aparece en la pantalla. Botones, toolbars, e inputs, todo es un View. En Flutter, el equivalente aproximado a un View es un Widget.

iOS

En iOS, la mayor parte de lo que se crea en la interfaz de usuario se hace utilizando objetos de vista, que son de la clase UIView. En Flutter, el equivalente aproximado a un UIView es un Widget.

Windows

La interfaz de programación de Windows combina APIs tradicionales y modernas basadas en C. La librería FFI le permite al lenguaje DART llamar a las librerías de C.

macOS

Flutter incluye a Cupertino, un conjunto de widgets para el lenguaje de diseño de iOS, el cual también es apropiado para macOS. Una alternativa es el paquete macos_ui.

Linux

La interfaz de programación de Linux está diseñada alrededor del lenguaje C. La librería FFI le permite al lenguaje DART llamar a librerías C y hacer uso de sus funciones y llamadas a sistema.
